🔷 Project Title

CIFAKE++: A Modified CNN2D Architecture with GAP and Dropout Layers for Robust Fake Image Identification

📖 Description

CIFAKE++ is a deep learning-based web application designed to detect whether an image is real or fake.
This project uses a Modified CNN2D architecture enhanced with Global Average Pooling (GAP) and Dropout layers to improve generalization and reduce overfitting.

The model achieves an accuracy of 95%, making it a reliable solution for fake image detection.

🚀 Features
🧠 Deep Learning-based image classification
📸 Upload and analyze images in real-time
⚡ Fast prediction using optimized CNN model
🎯 High accuracy (~95%)
🌐 Web interface using Flask

🛠️ Technologies Used
Programming Language: Python
Framework: Flask
Libraries: NumPy, TensorFlow / Keras (if used, add cheyyi)
Model: CNN2D (Modified with GAP + Dropout)

🧩 Model Architecture
Convolutional Layers (Feature Extraction)
Activation Functions (ReLU)
Pooling Layers
Global Average Pooling (GAP)
Dropout Layers (Overfitting reduction)
Fully Connected Layer
Output Layer (Real / Fake Classification)

📊 Accuracy
✅ Model Accuracy: 95%
📈 Optimized using dropout and GAP layers for better performance
